# BUUJ - The top 10 distribution
This is the decompilation project of the 10 years anniversary pokémon distribution

It builds the following ROM:
```
	3830c68bbe20406f56e1f4df54b827d0dbd77714  buuj.gba
```

Since the data has not been decompiled yet, you'll need a copy of the rom and name it `baserom.gba`

The game is simple, take all code from `/asm` and make its equivalent C code in `/src`

You will need to build gbagfx by yourself because I'm dumb and don't know how to make a proper Makefile
